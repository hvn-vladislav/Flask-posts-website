# Flask-posts-website

## What It Does

It's a web-based platform built with **Python (Flask), HTML, and CSS** that allows users to create, manage, and interact with content in the form of posts. The application functions as a lightweight social or blogging platform where users can publish their own posts and engage with content shared by others.

Each post can include a title, textual content, and images. Users have full control over their own posts and can interact with posts created by other users through a liking system.

---

## Features

* User authentication and session management
* Create posts with titles, text, and images
* Delete posts created by the current user
* Like posts published by other users
* Server-side rendering with Flask templates
* Simple and responsive interface using HTML and CSS

---

## Technologies Used

* **Python**
* **Flask**
* **SQLite**
* **HTML5**
* **CSS3**
* **Flask-Login**
* **Werkzeug Security**

## How to Run the Project

1. **Install Dependencies**
   Make sure Python is installed, then install the required libraries:

   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Application**
   Navigate to the project directory and start the Flask server:

   ```bash
   python main.py
   ```

3. **Open in Browser**
   Open your browser and go to:

   ```
   http://127.0.0.1:5000
   ```
